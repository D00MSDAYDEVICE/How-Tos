# HowTos, Scripts & Tools
This is a collection of useful things I've created or collected

[Guide](https://github.com/D00MSDAYDEVICE/How-Tos/blob/main/pi_zero_usb_camera_video_sync.md) to turning your Pi Zero 2W in to a "smart usb" drive to copy videos from a Blink camera system to your own server share.


Useful command scripts:
[Dockers](https://github.com/D00MSDAYDEVICE/How-Tos/blob/main/dockers.sh) - Colour coded list of dockers, stauts, size and ports
[Ports](https://github.com/D00MSDAYDEVICE/How-Tos/blob/main/ports.sh) - Colour coded list of listening ports
Place in these custom commands in: /usr/local/bin
chmod +x and the .sh suffix can be removed.
