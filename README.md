# HowTos, Scripts & Tools
This is a collection of useful things I've created or collected

[Guide](https://github.com/D00MSDAYDEVICE/How-Tos/blob/main/pi_zero_usb_camera_video_sync.md) to turning your Pi Zero 2W in to a "smart usb" drive to copy videos from a Blink camera system to your own server share.
<br>
See also [Video Vault](https://github.com/D00MSDAYDEVICE/video-vault) to pair with this for quick previews/playback.

Useful command scripts:<br>
[Dockers](https://github.com/D00MSDAYDEVICE/How-Tos/blob/main/dockers.sh) - Colour coded list of dockers, stauts, size and ports<br>
<br>
[Ports](https://github.com/D00MSDAYDEVICE/How-Tos/blob/main/ports.sh) - Colour coded list of listening ports, what's using them and PIDs<br>
Place in these custom commands in: /usr/local/bin<br>
chmod +x and the .sh suffix can be removed.
